# Marriage Portfolio — Itishree Lenka

A static, print-friendly marriage biodata website built with HTML, CSS, JavaScript and Bootstrap 5.

## Live Site

Hosted on GitHub Pages.

## Local Preview

Just open `index.html` in any modern browser.

## Tech

- HTML5 / CSS3 / Vanilla JavaScript
- Bootstrap 5 (via CDN)
- Bootstrap Icons (via CDN)
- Google Fonts: Cinzel, Great Vibes, Playfair Display, Poppins

## Features

- Elegant traditional theme (cream / maroon / gold)
- Fully responsive layout
- Print / Save-as-PDF support (A4, page-break aware)
- Subtle scroll-in animations
